How it works

Some key combination shortcuts are "blacklisted" to protect functionality. 'Ctrl + s', for example, is a traditional save shortcut. When a blacklisted shortcut is created you will be given a warning in the console like the one below, but the key will still be added.
Note that this will only show when in developer mode.

To view the warning message, open your browser's console then click the button below.
