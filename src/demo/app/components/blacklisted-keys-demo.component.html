How it works

Some key shortcuts are "blacklisted" to protect functionality. Ctrl + 's', for example, is a traditional save shortcut. When a blacklisted shortcut is created you will be given a warning in the console like the one below, but the key will still be added.

To view the warning message, enter the console and click the button below!
