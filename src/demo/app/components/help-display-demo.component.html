How it works

The help display demo is designed to show how the creation of any number of keybindings can be easily represented in a list (or table) format within some div. Saving an array of objects containing a keybinding combination and a description is the simple implementation of storing a help display for users using the system that the developer may require / want the user to be aware of.

The code below demonstrates how to construct a table that contains this information while building the keybindings and creating the listeners.
