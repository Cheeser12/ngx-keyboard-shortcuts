How it works

The directive is to make a shortcut for a button. Clicking the Show The Message button and an alert will show up. Click Ctrl + i and the same alert will show up.

If you look at the code provided it gives you the ability to call a message via the button click or by just using the directive selector.



     Button
[keyboardShortcut]="keyboardShortcutDef"
(click)="alertMessage()"
